# Fonts

Drop the Inter weights actually used (to keep app size down — plan/14) here
before building:

- `Inter-Regular.ttf`  (400)
- `Inter-Medium.ttf`   (500)
- `Inter-SemiBold.ttf` (600)

Download from https://rsms.me/inter/ or Google Fonts. These are declared in
`pubspec.yaml`; the build expects them present.
